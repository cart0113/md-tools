# Nested Headers Demo

This page demonstrates all six header levels and how they render with the theme.

## Second-Level Header

Paragraph text under a second-level header. This is the most common section
divider in documentation.

### Third-Level Header

Third-level headers are good for breaking up longer sections into digestible
subsections. They still appear in the sidebar when `subMaxLevel` is set high
enough.

#### Fourth-Level Header

Fourth-level headers work well for detailed breakdowns within a subsection.
Most documentation uses h1–h4 as its full range.

##### Fifth-Level Header

Fifth-level headers are rare in practice but useful for deeply structured
reference docs or specifications.

###### Sixth-Level Header

The smallest header level. Visually similar to bold body text in most themes.
